About Me

저는 실생활의 문제를 기술로 해결하는 것에 큰 보람을 느끼는 개발자입니다. 특히 Spring Boot와 React를 활용한 풀스택 개발에 강점을 가지고 있으며, 사용자 중심의 UI/UX 설계를 중요하게 생각합니다. 동양미래대학교 컴퓨터소프트웨어공학과 재학 중이며, Korea SGS에서의 인턴십을 통해 실무 경험을 쌓았습니다.

졸업 프로젝트로 Unity, Spring Boot, React를 결합한 게임 플랫폼 'Looper'를 개발하며 MCP 서버 기술과 AI 챗봇 시스템 구축 경험을 쌓았습니다.

이번 TradeVision 프로젝트는 제가 직접 주식 투자를 하며 느꼈던 불편함과 필요성을 바탕으로 기획했습니다. 초보 투자자도 쉽게 차트 분석을 배우고 활용할 수 있는 플랫폼을 만들고자 했으며, AI 기술을 접목하여 더 나은 투자 의사결정을 지원하는 것이 목표입니다.

문제 정의 (The Problem)

요즘 같은 고물가, 고환율 시대에 주식 투자를 하는 사람들이 대부분입니다. 하지만 많은 투자자들이 차트 분석 없이 단순히 가치투자나 감으로 매매를 하고 있습니다. 차트 분석이 무의미하다고 생각하는 사람들도 많지만, 제가 직접 투자하며 느낀 바로는 5일선 터치, 피보나치 되돌림 등의 기술적 분석이 상당히 유용한 지표가 되는 경우가 많았습니다. 특히 최근에는 프로그램 매매와 알고리즘 매매가 활성화되면서, 특정 기술적 지표에 따라 자동으로 매수/매도 주문이 체결되는 경우가 많습니다. 신기하게도 주요 지지선이나 저항선에서 딱 맞춰 거래량이 몰리는 현상을 자주 목격할 수 있습니다. 그러나 차트 분석 기법의 이름이나 적용 방법을 모르는 초보 투자자들은 이러한 기회를 놓치고 있으며, 어떤 시간봉에서 어떤 분석 기법을 사용해야 하는지에 대한 가이드도 부족한 상황입니다. 이러한 문제를 해결하기 위해 AI 기반의 차트 분석과 함께 학습 기능을 제공하는 통합 플랫폼이 필요하다고 판단했습니다.

아키텍처 / 설계 (Architecture)

사용자 친화적인 투자 분석 플랫폼을 위해 React 기반의 SPA(Single Page Application) 구조를 채택했습니다. 백엔드는 Spring Boot를 사용하여 RESTful API를 설계했으며, MariaDB를 통해 사용자 정보와 분석 이력을 관리합니다. 실시간 주식 데이터는 한국투자증권 OpenAPI를 연동하여 제공하고, 차트 이미지 분석에는 OpenAI GPT-4 Vision API를 활용했습니다. 봉 종류별 최적 매매기법 추천 로직은 백엔드에서 처리하며, 프론트엔드에서는 직관적인 UI/UX를 통해 초보자도 쉽게 사용할 수 있도록 설계했습니다. JWT 기반 인증을 통해 사용자별 분석 이력과 학습 진도를 관리하며, 학습 페이지는 별도 모듈로 구성하여 확장 가능한 구조를 만들었습니다.

시연 영상 (Demo Video)

TradeVision 프로젝트의 핵심 기능(실시간 차트 분석 및 결과 도출) 시연 영상입니다.

핵심 기능

AI 기반 실시간 차트 분석 시스템

사용자가 주식 코드를 입력하면 한국투자증권 OpenAPI를 통해 실시간 차트 데이터를 가져오거나, 직접 차트 이미지를 업로드할 수 있습니다. 사용자가 분석하고자 하는 봉 종류(1분봉/5분봉/15분봉/60분봉/일봉/주봉/월봉)를 선택하면, 분석 버튼을 누르기 전에 자동으로 알림창이 표시되어 해당 봉 종류에 최적화된 매매기법을 추천합니다. 예를 들어 1분봉을 선택하면 "스캘핑, 단기 RSI, 단기 이동평균선 크로스"를, 일봉을 선택하면 "피보나치 되돌림, 추세선 분석, 이동평균선(20, 60, 120), 지지/저항선"을 추천하는 방식입니다. 사용자가 추천 매매기법을 확인한 후 분석 버튼을 누르면 OpenAI GPT-4 Vision API가 차트 이미지를 분석하여 현재 패턴, 지지/저항선, 주요 이동평균선 위치 등을 파악하고, 매수/매도/관망 중 하나의 의견과 구체적인 근거를 제시합니다. 이를 통해 초보 투자자도 전문가 수준의 차트 분석 인사이트를 얻을 수 있습니다.

체계적인 기술적 분석 학습 시스템

차트 분석 용어와 기법을 모르는 초보 투자자를 위한 별도의 학습 페이지를 구현했습니다. 이동평균선, 볼린저밴드, MACD, RSI, 피보나치 되돌림 등 주요 기술적 분석 기법을 카테고리별로 분류하고, 각 기법의 개념, 계산 방법, 실제 차트 예시, 활용 전략을 상세히 설명합니다. 특히 각 분석 기법이 어떤 시간봉에서 효과적인지, 장단점은 무엇인지, 다른 지표와 어떻게 조합하면 좋은지까지 실용적인 정보를 제공합니다. React 기반의 인터랙티브한 UI를 통해 실제 차트 데이터에 각 지표를 적용해보며 학습할 수 있도록 구성했습니다. 사용자의 학습 진도는 데이터베이스에 저장되어 개인별 맞춤 학습 경로를 제공합니다.

회고 (Retrospective)

💡 배운 점 (Learnings)

OpenAI Vision API를 활용한 이미지 분석 프로세스와 프롬프트 엔지니어링의 중요성을 깊이 이해하게 되었습니다. 특히 차트 분석이라는 도메인 특화된 작업에서 AI에게 정확한 컨텍스트와 기대 출력 형식을 제공하는 것이 결과 품질에 큰 영향을 미친다는 것을 배웠습니다. 또한 실시간 금융 데이터를 다루며 API 호출 최적화와 에러 핸들링의 중요성을 체감했습니다. 사용자 중심 설계를 위해 복잡한 차트 분석 개념을 어떻게 직관적으로 전달할 것인가에 대해 많이 고민하며 UI/UX 설계 역량을 키울 수 있었습니다.

🚀 개선점 (Improvements)

향후 백테스팅 기능을 추가하여 과거 데이터로 AI의 분석 정확도를 검증하고 개선하고 싶습니다. 또한 사용자들의 실제 매매 결과를 익명으로 수집하여 분석 알고리즘의 성공률을 지속적으로 업데이트하는 피드백 루프를 구축하고 싶습니다. 기술적으로는 WebSocket을 활용한 실시간 차트 업데이트 기능과, 사용자별 맞춤형 알림 시스템(특정 기술적 패턴 발생 시 푸시 알림)을 추가하여 서비스의 실용성을 더욱 높이고 싶습니다. 마지막으로 React Testing Library와 Jest를 활용한 단위 테스트 및 통합 테스트를 작성하여 코드 안정성을 확보하고자 합니다.
